# AgendaPorf - Documento de Requisitos.
## Requisitos funcionais:
* RF1: Autenticação de usuários - Login dos usuários para acessar o app (Vendedor/Cliente)
* RF2: Registro de usuários - Cadastro de novos usuários (Vendedor/Cliente)
* RF3: Registro dos produtos - Cadastro de produtos para lojas (Vendedor)
* RF4: Compra dos produtos - Possíbilidade de comprar cortinas através do app (Cliente)

## Regras de negócios:
* RN1: Gerência de comunicação entre clientes e vendedores
* RN2: O vendedor precisa ter produtos
* RN3: O vendedor tem que estar vinculado à empresa
